# WAI bot.AI

WAI bot.AI by AiBuiltSales.com.

## Installation

### macOS
1. Download the latest **`.dmg`** from the [Releases page](https://github.com/digi-bulk-marketing/6a6071fb753c32f74ad8de4d/releases/latest).
2. Open the `.dmg` and drag **WAI bot.AI** into Applications.
3. Launch **WAI bot.AI** from Applications (first launch: right-click → Open if Gatekeeper warns).

## Updates

Download the newest installer from the [Releases page](https://github.com/digi-bulk-marketing/6a6071fb753c32f74ad8de4d/releases/latest) and reinstall.
