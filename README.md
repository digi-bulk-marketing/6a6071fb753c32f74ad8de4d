# WAI bot.AI

WAI bot.AI by AiBuiltSales.com.

## Installation

### macOS

**Pick the download matching your CPU chip:**

- **Apple Silicon (M1/M2/M3):** files containing `arm64` (e.g. `WAI-bot.AI-1.0.0-arm64.dmg` or `WAI-bot.AI-1.0.0-arm64-mac.zip`)
- **Intel:** files **without** `arm64` (e.g. `WAI-bot.AI-1.0.0.dmg` or `WAI-bot.AI-1.0.0-mac.zip`)

**Install and open (step-by-step):**

1. Download the correct file for your chip from the [Releases page](https://github.com/digi-bulk-marketing/6a6071fb753c32f74ad8de4d/releases/latest).
2. Open the `.dmg` (or unzip the `.zip`) and drag **WAI bot.AI** into `/Applications`.
3. If macOS blocks the app, open Terminal and run:

   ```bash
   sudo xattr -rd com.apple.quarantine "/Applications/WAI bot.AI.app"
   ```

4. Launch **WAI bot.AI** from Applications.

> If macOS shows a quarantine warning ("app is damaged" / "cannot be opened"), run the Terminal command above to unblock it, then launch again.
